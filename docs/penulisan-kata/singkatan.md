- _Prof._ = *prof*esor
- _S.E._ = *s*arjana *e*konomi
- _S.I.P._ = *s*arjana *i*lmu *p*olitik
- _S.K.M._ = *s*arjana *k*esehatan *m*asyarakat
- _S.Kom._ = *s*arjana *kom*puter
- _S.Sos._ = s*arjana *sos\*ial
- _Sp.A._ = *sp*esialis *a*nak
